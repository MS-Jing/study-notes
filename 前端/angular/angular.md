# 创建项目

+ 需要nodejs环境

+ 安装angular脚手架

  + ```npm
    cnpm install -g @angular/cli
    ```

+ 创建项目

  + ```bash
    ng new 项目名称
    #阻止自动下载安装相关依赖
    ng new 项目名称 --skip-install
    
    cnpm install
    ```

+ 项目启动

  + ```bash
    ng serve --open
    ```

+ 创建组件

  + ```bash
    ng g component components/news
    ```
